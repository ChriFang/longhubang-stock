# AI分析功能配置指南

## 🤖 功能简介

本系统集成了**智谱清言GLM-4大模型**，作为资深龙虎榜打板专家，为您提供专业的深度分析。

### AI分析功能包括：

1. **机会股票智能挖掘** 🎯（核心功能）
   - AI智能评分系统（满分100分）
   - 从所有龙虎榜股票中筛选次日大概率上涨的机会股票
   - 五大评分维度：
     * 买入资金含金量（0-30分）：识别顶级游资、多游资合力
     * 净买入额绝对值（0-25分）：真金白银的硬实力
     * 卖出方压力（0-20分）：主力了结vs散户离场
     * 机构共振（0-15分）：机构+游资联动（重点）
     * 其他加分项（0-10分）：买卖比、主力集中度、热门概念
   - Top股票详情对比
   - AI专家综合推荐
   - 导出评分表格和分析报告
   - **自动保存到历史记录数据库**

1.5. **历史分析记录查看** 📜
   - 查看过往的AI分析记录
   - 支持7/15/30/60/90天范围查询
   - 完整展示评分排名和AI分析
   - 支持回顾和对比历史分析
   - 可删除不需要的历史记录
   - 便于复盘和总结规律

2. **单股深度分析** 📈
   - 买入资金含金量评估（顶级游资 vs 普通游资）
   - 净买入额绝对值分析（真金白银 vs 小打小闹）
   - 卖出方压力分析（主力了结 vs 散户离场）
   - 机构共振分析（机构+游资联动）
   - 次日投资机会推演
   - 具体操作策略建议

3. **市场热点分析** 🔥
   - 当日市场主线和热点板块
   - 资金流向特征和偏好
   - 顶级游资动向追踪
   - 机构资金介入情况
   - 次日重点关注方向

4. **游资风格分析** 🎭
   - 游资等级评定
   - 操盘风格特点
   - 选股偏好和规律
   - 资金运作特点
   - 跟随注意事项

## 🚀 快速配置

### 第一步：获取智谱AI API Key

1. 访问 [智谱AI开放平台](https://open.bigmodel.cn/)
2. 注册/登录账号
3. 进入"API密钥"页面
4. 点击"创建API密钥"
5. 复制生成的API Key

### 第二步：配置环境变量

#### Windows用户：

1. 在项目根目录创建`.env`文件
2. 添加以下内容：
```
ZHIPU_API_KEY=你的API密钥
```
3. 保存文件

**快捷方式：** 运行 `setup_env.bat`

#### Linux/Mac用户：

1. 在项目根目录创建`.env`文件
```bash
touch .env
```

2. 编辑文件并添加：
```
ZHIPU_API_KEY=你的API密钥
```

**快捷方式：** 运行 `./setup_env.sh`

### 第三步：安装依赖

```bash
pip install -r requirements.txt
```

### 第四步：重启应用

```bash
streamlit run app.py
```

## 💡 使用技巧

### 1. 机会股票挖掘（推荐优先使用）
- 先查询获取龙虎榜数据
- 在"AI深度分析"标签页选择"🎯 机会股票挖掘"
- 选择要显示的Top N股票数量（建议10只）
- 点击"开始AI智能挖掘"
- 查看AI智能评分排名
- 重点关注：
  * 综合评分80分以上的股票
  * 机构共振评分高的股票（有✅标记）
  * 顶级游资参与2家以上的股票
  * 卖出压力评分高（压力小）的股票
- 查看Top 3详情和AI专家分析
- 导出评分表格和分析报告

### 2. 单股分析
- 先查询获取龙虎榜数据
- 在"AI深度分析"标签页选择"单股深度分析"
- 选择要分析的股票
- 点击"开始AI深度分析"
- AI将从多个维度给出专业分析

### 3. 市场热点分析
- 获取当日龙虎榜数据
- 选择"市场热点分析"
- AI将分析整体市场特征和投资机会

### 4. 游资风格分析
- 选择感兴趣的游资
- 设置分析周期（建议30天以上）
- AI将分析该游资的操盘特点和风格

## 📊 分析维度说明

### 买入资金"含金量"
- **顶级游资**: 章盟主、赵老哥等知名席位
- **一线游资**: 经常上榜，成功率较高的席位
- **普通游资**: 偶尔上榜的席位
- **单一主攻 vs 多家合力**: 分析资金集中度

### 净买入额分析
- **大额资金**: 通常500万以上，显示真金白银
- **中等资金**: 100-500万，常规操作
- **小额资金**: 100万以下，试探性买入

### 卖出压力分析
- **主力获利了结**: 大额席位卖出，需警惕
- **散户离场**: 小额卖出，筹码集中，看涨信号
- **机构减仓**: 机构席位卖出，需关注

### 机构共振
- **机构+游资同时买入**: 强烈看涨信号
- **纯游资操作**: 题材炒作，注意风控
- **纯机构操作**: 价值投资，相对稳健

## 🔧 故障排查

### 问题1：提示"AI分析功能未启用"
**解决方案**：
- 检查.env文件是否存在
- 检查API Key是否正确配置
- 重启Streamlit应用

### 问题2：分析失败，提示API错误
**解决方案**：
- 检查网络连接
- 确认API Key有效且有余额
- 查看智谱AI控制台是否有异常

### 问题3：分析速度慢
**解决方案**：
- GLM-4-Flash已是速度最快的模型
- 检查网络连接质量
- 可能是高峰期，稍后重试
