# 游资龙虎榜跟踪分析系统

基于Python和Streamlit开发的游资龙虎榜数据分析工具，提供实时的游资交易数据查询和分析功能。

## 功能特点

### 基础功能
- 📊 **实时数据查询**: 支持单日和日期范围查询
- 📈 **可视化分析**: 提供趋势图表和统计分析
- 🏦 **游资追踪**: 跟踪特定游资的操作记录
- 📋 **股票分析**: 分析股票受游资关注程度
- 💾 **数据缓存**: 智能缓存机制减少API调用

### 数据库功能
- 🗃️ **SQLite数据库**: 完整的历史数据存储
- 🔄 **自动数据持久化**: 查询数据自动保存到数据库
- 📊 **数据库管理**: 备份、统计和监控功能
- 🔍 **历史查询**: 支持历史数据回溯分析

### 高级分析功能
- 🎯 **游资胜率分析**: 计算游资历史操作胜率
- 🔍 **交易模式分析**: 识别游资操作习惯和偏好
- 🤖 **AI综合分析**: 智能评估游资投资价值
- 🔮 **趋势预测**: 基于游资活动的股票趋势预测
- 📊 **游资排名**: 多维度游资实力排名

### 🤖 AI深度分析功能（NEW！）
集成**DeepSeek 大模型**，作为资深龙虎榜打板专家，提供专业的投资分析：

- 🎯 **机会股票智能挖掘**（核心功能）:
  - 从所有龙虎榜股票中智能筛选次日大概率上涨的机会股票
  - AI智能评分系统（满分100分）：
    * 买入资金含金量评估（0-30分）：顶级游资识别、多游资合力
    * 净买入额绝对值分析（0-25分）：真金白银的硬实力
    * 卖出方压力评估（0-20分）：主力了结vs散户离场
    * 机构共振分析（0-15分）：机构+游资联动（重点关注）
    * 其他加分项（0-10分）：买卖比、主力集中度、热门概念
  - Top股票详情展示和对比
  - AI专家综合分析和操作建议
  - 支持导出评分表格和分析报告
  - **自动保存分析历史记录到数据库**

- 📜 **历史分析记录查看**:
  - 查看过往的AI分析记录
  - 支持按天数范围筛选（7/15/30/60/90天）
  - 完整保存评分排名、Top 3详情、AI分析报告
  - 支持回顾历史分析，便于复盘
  - 可删除不需要的记录

- 📈 **单股深度分析**: 
  - 买入资金"含金量"评估（顶级游资vs普通游资）
  - 净买入额绝对值分析（真金白银vs小打小闹）
  - 卖出方压力分析（主力了结vs散户离场）
  - 机构共振分析（机构+游资联动）
  - 次日投资机会推演
  - 具体操作策略建议

- 🔥 **市场热点分析**:
  - 当日市场主线和热点板块识别
  - 资金流向特征和偏好分析
  - 顶级游资动向追踪
  - 机构资金介入情况评估
  - 次日重点关注方向

- 🎭 **游资风格分析**:
  - 游资等级评定（顶级/一线/二线）
  - 操盘风格特点（激进/稳健）
  - 选股偏好和规律识别
  - 资金运作特点分析
  - 跟随注意事项建议

## 安装部署

### 1. 创建虚拟环境
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
```

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

### 3. 配置AI分析功能（可选但推荐）

#### 快速配置：
**Windows用户：** 运行 `setup_env.bat`  
**Linux/Mac用户：** 运行 `./setup_env.sh`

#### 手动配置：
1. 访问 [DeepSeek 开放平台](https://platform.deepseek.com/) 注册账号
2. 获取 API Key
3. 在项目根目录创建 `.env` 文件
4. 添加内容：
```
DEEPSEEK_API_KEY=你的API密钥
DEEPSEEK_BASE_URL=https://api.deepseek.com
```

详细配置步骤请查看：[AI_SETUP_GUIDE.md](AI_SETUP_GUIDE.md)

### 4. 运行应用
```bash
# 运行主应用（包含AI功能）
streamlit run app.py

# 或使用启动脚本
run.bat      # Windows
./run.sh     # Linux/Mac
```

## API接口说明

### 基本信息
- **接口地址**: https://www.stockapi.com.cn/v1/youzi/all
- **请求方式**: GET
- **更新频率**: 交易日下午5点40
- **速率限制**: 40次/秒
- **免费额度**: 1000次/天

### 请求参数
| 参数名 | 类型 | 是否必须 | 说明 |
|--------|------|----------|------|
| date | string | 是 | 查询日期 (YYYY-MM-DD) |

### 响应参数
```json
{
  "code": 20000,
  "msg": "success", 
  "data": [
    {
      "yzmc": "游资名称",
      "yyb": "营业部",
      "gpmc": "股票名称",
      "gpdm": "股票代码", 
      "mrje": "买入金额",
      "mcje": "卖出金额",
      "jlrje": "净流入金额",
      "rq": "日期",
      "gl": "概念"
    }
  ]
}
```

## 使用说明

### 基础功能
1. **单日查询**: 选择特定日期查看当天的游资上榜数据
2. **日期范围查询**: 选择开始和结束日期查看时间段内的数据
3. **数据过滤**: 设置最小金额过滤条件
4. **分析功能**: 
   - 趋势分析: 查看买卖金额变化趋势
   - 游资分析: 跟踪特定游资的操作记录
   - 股票分析: 分析股票受关注程度

### 🤖 AI深度分析
1. **查询数据**: 先使用基础功能获取龙虎榜数据
2. **选择分析类型**: 
   - 🎯 **机会股票挖掘**（推荐）：AI智能筛选次日大概率上涨的股票
   - 📜 **历史分析记录**：查看和回顾过往的AI分析记录
   - 单股深度分析：深度解析个股投资机会
   - 市场热点分析：把握市场主线和热点
   - 游资风格分析：了解游资操盘特点
3. **查看AI分析报告**: 获得专业的投资策略建议
4. **导出分析结果**: 下载评分表格和分析报告
5. **自动保存历史**: 分析结果自动保存到数据库，随时回顾
6. **结合实际操作**: AI建议仅供参考，需结合自身判断

### 最佳实践
- 每日盘后下载龙虎榜数据
- **优先使用"机会股票挖掘"找出高分股票**
- 关注机构共振和顶级游资参与的股票
- 使用AI分析市场热点
- 深度分析关注的个股
- 追踪明星游资动向
- **定期查看历史分析记录，总结AI推荐的准确率**
- 对比不同日期的分析结果，找出规律
- 建立投资决策笔记
- 导出分析报告供复盘使用

## 注意事项

### 数据使用
- 数据仅在交易日更新，非交易日无法获取新数据
- 请合理使用API，避免超过免费额度限制
- 建议在交易日下午5点40后获取最新数据

### AI分析
- AI分析基于 DeepSeek 聊天模型
- 分析结果仅供参考，不构成投资建议
- 需要配置有效的 API Key 才能使用
- AI 调用按量计费，成本以 DeepSeek 官方价格为准
- 请关注账户余额与调用限额

### 投资风险
- ⚠️ **股市有风险，投资需谨慎**
- 请结合自身风险承受能力做决策
- 建议设置止损止盈位
- 不要盲目跟风操作
